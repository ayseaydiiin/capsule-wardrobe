# Capsule Wardrobe Platform 👗

Full-stack MERN application for digital wardrobe management and outfit creation.

Capsule Wardrobe Platform is a web application that allows users to manage their clothing digitally, create outfit combinations, and interact with other users through social features.

The platform combines wardrobe management, outfit creation, and social interaction into a single modern application.

---

# 📌 About the Project

This project was developed using the **MERN stack (MongoDB, Express.js, React, Node.js)**.

Users can create their own digital wardrobe, build outfit combinations using a **drag & drop interface**, and share their outfits with other users.

The application also includes **authentication, messaging, notifications, and social interaction features**.

---

# 🚀 Features

### 👕 Digital Wardrobe
- Add, edit and delete clothing items
- Categorize clothes (tops, bottoms, shoes, accessories, etc.)
- Upload clothing images
- Filter items by category and color

### 🎨 Outfit Builder
- Drag & drop interface to create outfits
- Combine clothing pieces easily
- Save and share outfits

### 👥 Social Platform
- Share outfit posts
- Like and comment on posts
- Add friends
- Friends-only interactions

### 💬 Messaging
- Send messages between friends
- Floating messenger widget

### 🔔 Notifications
- Notification panel in the navigation bar
- Real-time notification updates

### 🌦 Weather Integration
- Outfit suggestions based on weather data

# 🏗 Architecture

The project follows a **full-stack MERN architecture**.

Frontend (React) communicates with the backend through REST APIs built with Express.js.  
Data is stored in MongoDB and managed using Mongoose models.

Authentication is implemented using **JWT tokens**, and passwords are securely hashed using **bcrypt**.
